# 🍌 Banana Chips Website

A clean, modern **Next.js + Tailwind CSS** website for a Banana Chips business.

## 🚀 Features
- Home, About, Products, Contact pages
- Responsive design with Tailwind CSS
- Navbar + Footer included
- Products grid (sample items)
- Contact form UI (no backend)

## 🛠 Installation

```bash
# install dependencies
npm install

# run development server
npm run dev

# build for production
npm run build
npm start
```

## 📂 Project Structure
- `pages/` → Website pages
- `components/` → Navbar, Footer, etc.
- `styles/` → Global styles
